# 📚 BookHub

BookHub is an AI-powered online bookstore that allows users to purchase physical books and E-books. The platform provides secure mobile money payments, AI-powered book recommendations, and instant E-book downloads.

## 🚀 Features

- User Registration & Login
- Browse and Search Books
- Shopping Cart & Checkout
- MTN MoMo & Airtel Money Payments
- E-book Downloads
- AI Chat Assistant
- Book Recommendations
- Admin Dashboard

## 🛠 Tech Stack

- React.js
- Tailwind CSS
- Node.js
- Express.js
- MongoDB Atlas
- JWT Authentication
- OpenAI/Gemini API
- Docker
- GitHub Actions
